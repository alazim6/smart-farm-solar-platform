# smart-farm-solar-platform
An Integrated Smart Platform for Farm Solar Energy Monitoring, Management &amp; Grid Integration – Software Engineering Course Project Report.

## Overview

This project report presents the planning and design of an integrated smart platform for farm solar energy monitoring, management and grid integration. The proposed system helps farm owners track energy production, monitor energy storage, transfer extra energy to the national grid, manage payments and request maintenance services. The report includes the project proposal, process model, Gantt chart, requirements, system design, UI design, Git workflow, Trello board, software testing and final conclusion.
